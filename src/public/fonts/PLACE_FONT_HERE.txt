下载字体
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. 访问: https://fonts.google.com/specimen/Pixelify+Sans
  
  2. 点击右上角 "Download family" 按钮
  
  3. 解压 ZIP 文件
  
  4. 进入 static 文件夹
  
  5. 复制 PixelifySans-Regular.ttf
  
  6. 粘贴到这个文件夹（/public/fonts/）


安装后
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. 在 VSCode 终端按 Ctrl+C 停止服务器
  
  2. 输入: npm run dev
  
  3. 刷新浏览器
  
  4. 完成！字体应该正常显示了


检查清单
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ☐ 文件是否在 /public/fonts/ 文件夹中？
  ☐ 是否重启了开发服务器？
  ☐ 是否清除了浏览器缓存？
