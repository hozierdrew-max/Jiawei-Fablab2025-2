          像素字体安装步骤（3步搞定）
═══════════════════════════════════════════════════════

第一步：下载字体
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 访问：https://fonts.google.com/specimen/Pixelify+Sans
2. 点击右上角 "Download family" 蓝色按钮
3. 下载 ZIP 文件到电脑


第二步：解压并找到字体文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 右键点击下载的 ZIP 文件 → 解压全部
2. 打开解压后的文件夹
3. 进入 "static" 文件夹
4. 找到 "PixelifySans-Regular.ttf" 文件
5. 复制这个文件（右键 → 复制）


第三步：放入项目文件夹
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 打开项目文件夹
2. 进入 public → fonts 文件夹
3. 粘贴字体文件到这里
4. 重启开发服务器（在VSCode终端按Ctrl+C，然后npm run dev）
5. 刷新浏览器，完成！


最终位置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
你的项目/
└── public/
    └── fonts/
        └── PixelifySans-Regular.ttf  ← 文件应该在这里


═══════════════════════════════════════════════════════

只需要 1 个文件：PixelifySans-Regular.ttf

如果想要更多字重（可选）：
- PixelifySans-Medium.ttf
- PixelifySans-SemiBold.ttf
- PixelifySans-Bold.ttf
