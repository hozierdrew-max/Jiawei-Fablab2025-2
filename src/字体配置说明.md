# 🎮 字体配置说明

## 📋 方案概述

本项目使用**本地字体方案**，确保完全离线可用，不依赖任何在线服务。

---

## ✅ 配置详情

### 1. 字体加载方式

**仅使用本地字体文件**
- ✅ 完全离线可用
- ✅ 加载速度最快
- ✅ 不需要网络连接
- ✅ 不依赖第三方服务
- ⚠️ 需要手动下载和放置字体文件

### 2. CSS 配置

在 `/styles/globals.css` 文件中配置：

```css
/* Local Font - Pixelify Sans */
@font-face {
  font-family: 'Pixelify Sans';
  src: url('/fonts/PixelifySans-Regular.ttf') format('truetype');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

/* 支持多种字重 */
@font-face {
  font-family: 'Pixelify Sans';
  src: url('/fonts/PixelifySans-Medium.ttf') format('truetype');
  font-weight: 500;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'Pixelify Sans';
  src: url('/fonts/PixelifySans-SemiBold.ttf') format('truetype');
  font-weight: 600;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'Pixelify Sans';
  src: url('/fonts/PixelifySans-Bold.ttf') format('truetype');
  font-weight: 700;
  font-style: normal;
  font-display: swap;
}
```

### 3. 字体文件夹

已创建 `/public/fonts/` 文件夹，用于存放字体文件。

## 🎯 使用方法

### 下载字体

**字体名称**: **Pixelify Sans**  
**下载地址**: https://fonts.google.com/specimen/Pixelify+Sans

### 需要的文件

**最简方案**（只需1个文件）:
```
PixelifySans-Regular.ttf
```

**完整方案**（可选，支持不同粗细）:
```
PixelifySans-Regular.ttf     （必需）
PixelifySans-Medium.ttf      （可选）
PixelifySans-SemiBold.ttf    （可选）
PixelifySans-Bold.ttf        （可选）
```

### 放置位置

```
你的项目文件夹/
└── public/
    └── fonts/
        └── PixelifySans-Regular.ttf  ← 放这里
```

### 安装步骤

1. 访问 https://fonts.google.com/specimen/Pixelify+Sans
2. 点击右上角 "Download family" 按钮
3. 解压 ZIP 文件
4. 进入 `static` 文件夹
5. 复制 `PixelifySans-Regular.ttf`
6. 粘贴到项目的 `/public/fonts/` 文件夹
7. 重启开发服务器

```bash
# 在VSCode终端中
# 按 Ctrl+C 停止服务器
npm run dev  # 重新启动
```

## 🔧 工作原理

### 字体加载流程

```
浏览器启动
  ↓
读取 CSS 中的 @font-face 配置
  ↓
尝试从 /public/fonts/ 加载字体文件
  ↓
如果文件存在
  → 使用本地字体 ✅
  ↓
如果文件不存在
  → 使用系统默认 sans-serif 字体 ⚠️
```

### 字体显示策略 (font-display: swap)

- 立即显示备用字体
- 字体加载完成后自动切换
- 避免 FOIT（Flash of Invisible Text）
- 优化用户体验

## 🆘 常见问题

### Q: 我下载了字体，放到哪里？

**A**: 放到 `/public/fonts/` 文件夹中

在 VSCode 中：
```
展开 public 文件夹
  → 展开 fonts 文件夹
    → 拖拽字体文件到这里
```

---

### Q: 文件名必须一致吗？

**A**: 是的！必须完全一致（包括大小写）

✅ 正确: `PixelifySans-Regular.ttf`  
❌ 错误: `pixelifysans-regular.ttf`  
❌ 错误: `Pixelify Sans Regular.ttf`

---

### Q: 字体文件很大吗？

**A**: 不大

`PixelifySans-Regular.ttf` 大约 60-80 KB  
非常小，不会影响网站性能。

---

### Q: 下载后字体还是不显示？

**A**: 检查清单

- [ ] 文件名是否完全一致？
- [ ] 文件是否在 `/public/fonts/` 文件夹中？
- [ ] 是否重启了开发服务器？
- [ ] 是否清除了浏览器缓存？

**调试步骤**：

1. 按 `F12` 打开浏览器开发者工具
2. 切换到 "Network"（网络）标签
3. 刷新页面
4. 搜索 "PixelifySans"
5. 查看字体文件是否成功加载

**查看错误信息**：

1. 按 `F12` 打开浏览器开发者工具
2. 切换到 "Console"（控制台）标签
3. 查看是否有红色错误信息
4. 常见错误：
   - `Failed to load resource` - 文件路径不正确
   - `404 Not Found` - 文件不存在

## 💡 优势说明

### 为什么使用本地字体？

| 特性 | 本地字体 | 在线字体 |
|------|---------|---------|
| 加载速度 | ⚡ 最快 | 🐌 较慢 |
| 离线可用 | ✅ 是 | ❌ 否 |
| 网络要求 | ❌ 不需要 | ✅ 需要 |
| 隐私性 | ✅ 完全本地 | ⚠️ 第三方CDN |
| 稳定性 | ✅ 不依赖外部 | ⚠️ 依赖外部服务 |
| 配置难度 | 📥 下载1个文件 | ✅ 无需配置 |

### 性能对比

**本地字体加载时间**：
- 首次加载：< 50ms
- 后续访问：< 10ms（浏览器缓存）

**在线字体加载时间**：
- 首次加载：200-500ms（取决于网络）
- 后续访问：100-200ms（CDN缓存）
- 离线：❌ 无法加载
